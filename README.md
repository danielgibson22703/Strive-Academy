# StriveAcademy

A web platform for purchasing professional coaching services for the 2D fighting game **Guilty Gear Strive**.

## Overview

StriveAcademy is a dedicated website designed to connect players with an experienced coach who can help improve their competitive gameplay, tech,  skills, and overall mastery of Guilty Gear Strive.

## Features

- **Coach Profile**: Explore a detailed profile of the coach
- **Easy Booking**: Streamlined coaching session purchase process
- **Secure Payments**: Safe transaction handling for coaching services
- **User-Friendly Interface**: Clean, responsive design for all devices
